Text Fields

`.tonic-field` groups labels, inputs, and supporting text. Inputs inherit spacing, borders, and focus ring.

Selects & Validation

Select inputs have custom indicators. Apply `aria-invalid="true"` or `.is-invalid` for error state.

Checkboxes & Radios

Inputs use pure CSS to render checks and indeterminate states. Labels align with doc utilities.
