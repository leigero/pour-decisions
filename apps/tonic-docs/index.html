Spacing Scale

Tokens follow quarter-rem increments. Utilities map directly to these values.

Token Value Usage
$space-000 0 Reset / hard edge
$space-025 0.25rem Micro spacing, icon alignment
$space-050 0.5rem Pill buttons, compact gaps
$space-075 0.75rem Input vertical padding
$space-100 1rem Paragraph rhythm, button padding
$space-150 1.5rem Stack default gap
$space-200 2rem Section padding
$space-300 3rem Hero padding, modal gutters
